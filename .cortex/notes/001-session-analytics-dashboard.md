ทำ dashboard แสดง analytics ของการใช้งาน Claude Code sessions

- session duration เฉลี่ยต่อ project
- จำนวน sessions ต่อวัน/สัปดาห์
- tools ที่ใช้บ่อยที่สุด
- project ไหนใช้เยอะสุด
- token usage estimate (ถ้าดึงได้)
- timeline view ของ sessions ทั้งหมด

เหมาะกับ user ที่อยากรู้ว่าใช้เวลากับ project ไหนมากสุด
