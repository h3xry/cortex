บันทึก terminal output ของแต่ละ session เป็นไฟล์

- auto-save transcript เมื่อ session จบ
- export เป็น .md หรือ .txt
- search ย้อนหลังใน transcript เก่า
- link transcript กับ note ได้ (เช่น "session นี้ทำตาม note #003")
- replay mode แบบ asciinema

ตอนนี้ output buffer แค่ 100KB แล้วหายตอน restart — ต้องมี persistence
