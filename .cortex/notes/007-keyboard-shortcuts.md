Global keyboard shortcuts สำหรับ power users

- Cmd+K → global search (command palette)
- Cmd+1-9 → switch project
- Cmd+T → new session
- Cmd+W → kill session
- Cmd+N → new note
- Cmd+Shift+P → switch tab (Terminal/Files/Changes/Notes)
- Cmd+, → settings
- ? → show shortcut cheatsheet

ทำแบบ VS Code command palette — พิมพ์ command แล้ว execute
