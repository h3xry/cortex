- search notes ทุก project พร้อมกัน
- search ใน file content ของ project (เหมือน grep แต่ผ่าน UI)
- search ใน session transcripts (ถ้ามี feature 002)
- filter ตาม project, tag, date range
- global shortcut เปิด search (Cmd+K)
